# ShopEase

ShopEase is a simple e-commerce web application built with **React**.  
Users can browse products, filter by categories, view product details, and manage a shopping cart.

---

## Features

- Browse products fetched from [Fake Store API](https://fakestoreapi.com/)  
- Filter products by category  
- View product details on a separate page  
- Add, remove, increment, and decrement products in the cart  
- Cart state is managed using **React Context API**  
- Cart is persisted in **localStorage**  
- Responsive design for desktop and mobile  

---

## Tech Stack

- **Frontend:** React.js  
- **State Management:** React Context API  
- **API:** Axios for fetching products from Fake Store API  
- **Routing:** React Router DOM  
- **Styling:** Bootstrap 5
